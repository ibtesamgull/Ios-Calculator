A visually striking, fully responsive web-based calculator. This project moves away from flat design, implementing a modern **Neumorphic (Soft UI)** aesthetic to create a tactile, realistic user experience directly in the browser. 

---

## 📸 Interface Showcase

> **Design Philosophy:** Combining a sleek dark mode background with carefully crafted CSS drop-shadows to create extruded, pressable buttons. The cyan typography ensures high contrast and readability for mathematical operators.

## ✨ Key Features

*   **Tactile Neumorphism:** Utilizes precise CSS `box-shadow` techniques (combining light and dark shadows) to give the illusion of physical, 3D buttons resting on a matte surface.
*   **High-Contrast Dark Theme:** A deep, slate-grey canvas paired with vibrant cyan/teal accents reduces eye strain while maintaining a sharp, professional look.
*   **Fluid Grid Layout:** Structured intuitively for ease of use, featuring an extended `=` button for seamless calculation flow.
*   **Lightweight & Fast:** Built entirely without heavy frameworks. Just pure, clean front-end code[cite: 4].

---

## 🛠️ Technical Architecture

This application is built with a strict separation of concerns, keeping the markup structural and the styling modular[cite: 4]:

*   **`index.html`**: Contains the semantic HTML skeleton and grid layout for the calculator interface[cite: 4].
*   **`style.css`**: Powers the visual engine, housing the Neumorphic shadow rendering, flexbox/grid alignments, and typography styling[cite: 4].
